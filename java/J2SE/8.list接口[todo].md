## ArrayList
- 遍历和迭代
``` java
Collection<String> c = new ArrayList<String>();
c.add("a");
c.add("b");
c.add("c");

// 使用iterator循环迭代
for(Iterator<String> i = c.iterator(); i.hasNext();) {
    System.out.println(i.next());
}

// 推荐这种for遍历语法糖
for (String word : c) {
    System.out.println(word);
}
```

- iterable接口
``` java
public interface Iterator<E> {
    boolean hasNext();
    E next();
    void remove();
}
```
